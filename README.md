# Blackvale Holdings

A modern, minimal, and high-end digital presence for **Blackvale Holdings, LLC** — a private investment and acquisition firm focused on long-term value creation.

## Overview

This static site is designed to reflect the prestige, discretion, and professionalism of the Blackvale Holdings brand. Built with Vite and deployed via Vercel, the site emphasizes elegance and trust with a sleek black, gold, and white theme.

## Tech Stack

- **Frontend**: HTML, CSS, Vite  
- **Deployment**: Vercel  
- **Domain**: [blackvaleholdings.com](https://blackvaleholdings.com)

## Features

- Lightning-fast static delivery  
- Fully responsive design  
- Custom professional logo  
- SEO-ready foundation  

## Setup Instructions

To run locally:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
```
